Basic Integration

Add this single line to your HTML to embed the widget:

<script src="http://localhost:3001/embed.js"></script>

That's it! The widget will automatically initialize with default settings.

Custom Container Integration

For more control over widget placement, you can specify a custom container:

<div id="my-chat-widget"></div> <script> const script = document.createElement('script'); script.src = 'http://localhost:3001/widget/matrix-chat-widget.iife.js'; script.onload = function() { fetch('http://localhost:3001/api/config') .then(response => response.json()) .then(config => { window.MatrixChatWidget.init(config, 'my-chat-widget'); }); }; document.head.appendChild(script); </script>

Production Deployment

For production, update your server configuration:

# In config/config.yaml server: cors_origins: - "https://your-website.com" - "https://www.your-website.com" widget: brand_color: "#your-brand-color" title: "Your Support Team" subtitle: "We're here to help!"

Then update the script URL to point to your production server.
