First Visit Test

Action: Click the chat button and start your first conversation.

  • Fill out the form with your name, email, and message
  • Click "Send Message" to start the chat
  • Send a few test messages in the chat
Expected: New user experience with welcome messages. No "Welcome back" indicator should appear.

Session Storage Test

Action: Close the chat modal and reopen it.

Expected: Form should be pre-filled with your previous details. Chat history should be maintained.

Page Refresh Test

Action: Refresh this page (F5 or Ctrl+R).

Expected: After page refresh, opening the chat should show "Welcome back!" message with conversation count. Previous form data should be pre-filled.

History Loading Test

Action: After refresh, start a new conversation.

Expected: Should show "Loading conversation history..." indicator, then display your previous messages. A "Previous conversation history loaded" indicator should appear.

Browser Session Test

Action: Close this browser tab, open a new tab, and navigate back to this page.

Expected: Chat persistence should work exactly like after page refresh. All previous conversation data should be available.

Fresh Start Test

Action: Use the "Start fresh conversation" button in the welcome message.

Expected: Storage should be cleared, form should be empty, and you should get the new user experience.

Storage Cleared Test

Action: Use the "Clear All Storage" button above, then open the chat.

Expected: Should behave like a completely new user. No pre-filled data, no welcome back message.
