Matrix Chat Support Widget - Integration Example

This page demonstrates how to integrate the Matrix Chat Support Widget into any website.

Note: Make sure your Matrix Chat Support server is running on localhost:3001 and properly configured before testing this example.

How it works

The widget is embedded using a simple script tag that:

  1. Loads the widget configuration from your server
  2. Downloads the widget JavaScript bundle
  3. Initializes the chat widget on your page

Integration Code

Add this single line to any webpage where you want the chat widget:

<script src="http://localhost:3001/embed.js"></script>

Custom Container (Optional)

By default, the widget creates its own container. You can also specify a custom container:

<div id="my-chat-widget"></div> <script> // Load the widget script first const script = document.createElement('script'); script.src = 'http://localhost:3001/widget/matrix-chat-widget.iife.js'; script.onload = function() { // Fetch configuration fetch('http://localhost:3001/api/config') .then(response => response.json()) .then(config => { if (window.MatrixChatWidget) { window.MatrixChatWidget.init(config, 'my-chat-widget'); } }); }; document.head.appendChild(script); </script>

Production Deployment

For production deployment:

  1. Replace localhost:3001 with your actual domain
  2. Ensure HTTPS is enabled
  3. Configure CORS origins in your config.yaml
  4. Set up proper SSL certificates

Testing the Widget

Click the chat button to test the widget functionality. Note that you'll need:

  • A valid Matrix homeserver URL
  • A valid access token for a Matrix user/bot
  • Network access to the Matrix server
